# Bus Ticket Booking System

## 📌 Description
Spring Boot backend application for managing bus ticket data.

## 🚀 Features
- Get buses by office ID
- Global exception handling
- Input validation

## 🛠 Tech Stack
- Java
- Spring Boot
- Spring Data JPA
- ModelMapper

## 🔥 API

GET /api/offices/buses?officeId=1

## ⚠️ Error Handling
- 400 → Validation errors
- 404 → Resource not found
- 500 → Internal errors
